## macOS setup

1. Install homebrew

<SingleCodeBlock code='/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"' />

2. Install node

<SingleCodeBlock code="brew install node" />

3. Setup npm permissions

<SingleCodeBlock code="sudo chown -R $(whoami) $(npm config get prefix)/{lib/node_modules,bin,share}" />

4. Install pnpm

<SingleCodeBlock code="npm i -g pnpm" />

5. mysql

   1. Download and install

      - [mysql community server](https://dev.mysql.com/downloads/mysql)
      - [mysql shell](https://dev.mysql.com/downloads/shell)

   2. Setup

      - <SingleCodeBlock title=".zprofile alias" code="echo 'alias mysql=/usr/local/mysql/bin/mysql' >> ~/.zprofile" />
      - <SingleCodeBlock title="Shell login" code="mysql -u root -p" />
      - <SingleCodeBlock title="Create user" code="CREATE USER 'username'@'localhost' IDENTIFIED BY 'password';" />
      - <SingleCodeBlock title="Create database" code="CREATE DATABASE db_name;" />
      - <SingleCodeBlock title="Use database" code="USE db_name;" />
      - <SingleCodeBlock title="Grant permissions for created user" code="GRANT ALL PRIVILEGES ON *.* TO 'username'@'localhost';" />
